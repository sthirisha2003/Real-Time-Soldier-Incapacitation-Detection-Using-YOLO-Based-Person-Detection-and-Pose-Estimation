This project implements an AI-based Human Posture Detection System using computer vision and machine learning techniques. The system detects humans in images or video streams using the YOLO (You Only Look Once) object detection algorithm and analyzes posture by extracting skeletal keypoints.

The detected posture data is then processed to classify different body positions using a machine learning model. This system can be applied in health monitoring, fitness tracking, workplace ergonomics, and surveillance applications.
